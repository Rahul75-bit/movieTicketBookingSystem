# 🎬 Movie Ticket Booking System

A Full Stack Movie Ticket Booking Application built using **React**, **Spring Boot**, and **MySQL**.

---

## 🚀 Features

- User Authentication
- Movie Listing
- Seat Booking
- Responsive UI
- REST API Integration
- Admin Management
- Movie Image Upload
- Booking Management

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Bootstrap
- Axios
- Vite

### Backend
- Spring Boot
- Spring Data JPA
- Hibernate
- REST APIs

### Database
- MySQL
